# Pythonパッケージテンプレート
本プロジェクトはパッケージングできるpythonプロジェクトを作成するときに使用するテンプレートです。

## Build
以下にてビルドを実施

```
./build.sh
```

## Install
以下にてインストールを実施

```
pip install python_package_project_template-0.0.1-py3-none-any.whl
```

## How to use

```
from example_package import example
example.add_one(1)
```
